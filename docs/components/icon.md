# 图标 Icon

## 代码演示

<ClientOnly>
  <demo-icon />
</ClientOnly>

## API

```vue
<f-icon :name="name" />
```

| 参数 | 说明     | 类型   | 默认值 | 可选值                                                                          |
| ---- | -------- | ------ | ------ | ------------------------------------------------------------------------------- |
| name | 图标名称 | string | -      | `info`/`error`/`loading`/`setting`/`up`/`down`/`left`/`right`/`download`/`like` |
