# fun-ui

> A UI toolkit base on Vue 2.

## Install

```shell
npm install fun-ui

# or

yarn add fun-ui
```

## Quick Start

```vue
<template>
  <div id="app">
    <f-button>Click me</f-button>
  </div>
</template>

<script>
import { FButton } from "fun-ui";
import "fun-ui/dist/fun-ui.css";

export default {
  name: "app",
  components: { FButton },
};
</script>
```
