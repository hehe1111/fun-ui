> A UI toolkit base on Vue 2.

## Install

```shell
npm install fun-ui

# or

yarn add fun-ui
```

## Quick Start

```js
import Vue from "vue";
import funUI from "fun-ui";
import "fun-ui/dist/fun-ui.css";
Vue.use(funUI);
```
